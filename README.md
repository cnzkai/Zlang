# 智语言（ZLang）开发者指南

## 一、语言简介

**核心特点**：
- 中文关键字，降低入门门槛
- 缩进表示代码块，无需大括号
- 变量类型自动推断，无需声明类型
- 数值类型动态选择最合适的表示（整数/浮点）

**代码风格**：
- 使用 4 空格缩进表示代码块
- 用冒号 `:` 开始代码块
- 每个语句独占一行，无需分号
- 变量和函数命名支持中文
- 单行注释用 `//`

```z
阶乘(n):
    如果 n <= 1:
        返回 1
    返回 n * 阶乘(n - 1)
```

## 二、基本类型

| 类型 | 关键字 | 说明 | 示例 |
|------|--------|------|------|
| 数值 | `数值` | 整数或浮点数 | `42`、`3.14` |
| 文本 | `文本` | UTF-8 字符串 | `"你好"` |
| 字节 | `字节` | 8位数据 | 字节数组 |
| 逻辑 | `逻辑` | 真/假 | `真`、`假` |
| 指针 | `指针` | 内存地址 | `&变量` |
| 数组 | `数组<类型>` | 动态数组 | `[1, 2, 3]` |
| 结构 | `结构` | 自定义数据体 | `{x: 10, y: 20}` |
| 空值 | `空` | 空引用 | `空` |

## 三、变量

### 3.1 自动推断（推荐）

直接赋值，类型自动推断：

```z
年龄 = 10           // 数值
姓名 = "张三"       // 文本
已婚 = 假           // 逻辑
数据 = [1, 2, 3]    // 数组
坐标 = {x: 10, y: 20}  // 结构
```

### 3.2 空值声明

用类型关键字声明未初始化变量，初始值为空：

```z
数值 年龄           // 初始值为空
文本 姓名           // 初始值为空
年龄 = 10           // 后续赋值
```

### 3.3 重新赋值

变量可以重新赋值，覆盖原有值：

```z
重用变量 = 10
重用变量 = 20       // 现在值是 20
```

## 四、运算符

### 4.1 算术运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法 / 字符串拼接 | `1 + 2` → 3，`"a" + "b"` → "ab" |
| `-` | 减法 | `5 - 3` → 2 |
| `*` | 乘法 | `4 * 5` → 20 |
| `/` | 除法 | `10 / 2` → 5 |
| `%` | 取模 | `17 % 5` → 2 |
| `-` | 负号 | `-10` |

### 4.2 比较运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `==` | 等于 | `1 == 1` → 真 |
| `!=` | 不等于 | `1 != 2` → 真 |
| `>` | 大于 | `5 > 3` → 真 |
| `<` | 小于 | `3 < 5` → 真 |
| `>=` | 大于等于 | `5 >= 5` → 真 |
| `<=` | 小于等于 | `3 <= 5` → 真 |

### 4.3 逻辑运算符

| 运算符 | 中文写法 | 说明 | 示例 |
|--------|----------|------|------|
| `&&` | `且` | 逻辑与 | `真 且 假` → 假 |
| `\|\|` | `或` | 逻辑或 | `真 或 假` → 真 |
| `!` | `非` | 逻辑非 | `!假` → 真 |

### 4.4 赋值运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `=` | 赋值 | `x = 10` |
| `+=` | 加后赋值 | `x += 5` 等同 `x = x + 5` |

### 4.5 指针运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `&` | 取地址 | `地址 = &变量` |
| `*` | 解引用 | `值 = *地址` |

**中文函数形式（等价）**：

| 函数 | 说明 | 示例 |
|------|------|------|
| `取地址(变量名)` | 返回变量的内存地址 | `地址 = 取地址(变量)` |
| `解引用(指针名)` | 读取指针指向的值 | `值 = 解引用(地址)` |

> 注：运算符形式和函数形式完全等价，可自由选择使用。

### 4.6 位运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `&` | 位与 | `0xFF & 0x0F` → 0x0F |
| `\|` | 位或 | `0xF0 \| 0x0F` → 0xFF |
| `^` | 位异或 | `0xFF ^ 0x0F` → 0xF0 |
| `~` | 位取反 | `~0x00` → 0xFFFFFFFFFFFFFFFF |

### 4.7 运算符优先级（从低到高）

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 1 | `=` `+=` | 赋值 |
| 2 | `或` `||` | 逻辑或 |
| 3 | `且` `&&` | 逻辑与 |
| 4 | `==` `!=` | 相等比较 |
| 5 | `>` `<` `>=` `<=` | 大小比较 |
| 6 | `|` | 位或 |
| 7 | `^` | 位异或 |
| 8 | `&` | 位与 |
| 9 | `+` `-` | 加减 |
| 10 | `*` `/` `%` | 乘除取模 |
| 11 | `!` `-` `~` `&`(取地址) `*`(解引用) | 一元运算 |
| 12 | `.` | 成员访问 |
| 13 | `[]` | 数组索引 |
| 14 | `()` | 函数调用 |

用括号可以改变优先级：`(2 + 3) * 4` → 20

## 五、控制流

### 5.1 条件语句

```z
如果 条件:
    ...

如果 条件:
    ...
否则:
    ...

如果 条件1:
    ...
否则如果 条件2:
    ...
否则:
    ...
```

支持任意深度嵌套：

```z
如果 50 > 0:
    如果 50 > 25:
        如果 50 > 75:
            条件结果 = 1
        否则:
            条件结果 = 2
```

### 5.2 计次循环

```z
计次循环 i = 0 到 5:
    输出("%d\n", i)    // 输出 0, 1, 2, 3, 4
```

- `i` 从起始值开始，到结束值之前停止（不包含结束值）
- 循环变量 `i` 可以在循环体内使用

### 5.3 条件循环

```z
条件循环 条件:
    ...
```

条件为真时持续执行：

```z
计数 = 0
条件循环 计数 < 3:
    计数 = 计数 + 1    // 执行 3 次
```

### 5.4 跳出与跳过

```z
跳出    // 立即退出当前循环
跳过    // 跳过本次循环，继续下一次
```

示例——跳出循环：

```z
计次循环 j = 0 到 20:
    如果 j == 5:
        跳出
    值 = j         // 值最终为 4
```

示例——跳过偶数：

```z
求和 = 0
计次循环 k = 0 到 10:
    如果 k % 2 == 0:
        跳过
    求和 = 求和 + k   // 只加奇数：1+3+5+7+9 = 25
```

## 六、函数

### 6.1 定义函数

用 `函数` 关键字定义，参数和返回值类型自动推断：

```z
加法(a, b):
    返回 a + b
```

### 6.2 调用函数

```z
结果 = 加法(3, 5)       // 结果 = 8
```

### 6.3 递归

```z
斐波那契(n):
    如果 n <= 1:
        返回 n
    返回 斐波那契(n - 1) + 斐波那契(n - 2)

阶乘(n):
    如果 n <= 1:
        返回 1
    返回 n * 阶乘(n - 1)
```

### 6.4 返回值

- `返回` 语句返回值并结束函数
- 函数无返回值时返回 `空`

```z
返回数值():
    返回 42

返回文本(n):
    返回 转文本(n)
```

### 6.5 主函数

程序入口是 `主函数`：

```z
主函数():
    输出("Hello!\n")
    退出(0)
```

## 七、数组

数组是动态长度的序列，索引从 0 开始。

### 7.1 创建数组

```z
整数数组 = [10, 20, 30, 40, 50]
二维数组 = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
空数组 = []
```

### 7.2 访问与修改

```z
值 = 整数数组[0]       // 读取 → 10
整数数组[0] = 999      // 修改
```

支持变量索引和表达式索引：

```z
索引值 = 1
值 = 整数数组[索引值]          // 20
值 = 整数数组[索引值 + 1]      // 30
```

二维数组用连续 `[]` 访问：

```z
值 = 二维数组[1][1]    // 5
```

### 7.3 追加元素

```z
数组 = [1, 2, 3]
追加(数组, 4)           // 数组变为 [1, 2, 3, 4]
```

### 7.4 获取长度

```z
取长度(整数数组)        // 返回元素个数
```

### 7.5 遍历数组

```z
数组求和 = 0
计次循环 idx = 0 到 取长度(整数数组):
    数组求和 = 数组求和 + 整数数组[idx]
```

## 八、结构体

结构体是自定义的数据集合，包含命名成员。

### 8.1 定义结构体类型

```z
结构 点类型{
    x: 数值
    y: 数值
}
```

### 8.2 创建实例

```z
点A = {x: 100, y: 200}
```

也可以不定义类型，直接创建：

```z
人物 = {name: "悟空", age: 500, city: "比奇省"}
```

### 8.3 访问与修改成员

```z
点A.x                    // 访问 → 100
人物.age = 501           // 修改
人物.city = "花果山"     // 修改
```

## 十、字符串操作

### 10.1 字符串拼接

用 `+` 拼接字符串：

```z
结果 = "你好" + "世界"          // "你好世界"
结果 = "智" + "语" + "言"       // "智语言"
结果 += "测" + "试"             // 追加拼接
结果 = "你好，" + 结果          // 前置拼接
```

### 10.2 取长度

```z
取长度("你好")          // 6（UTF-8 字节数）
取长度("智语言")        // 9
取长度("")              // 0
```

### 10.3 比较

```z
比较("abc", "abc")      // 0（相等）
比较("abc", "abd")      // <0（小于）
比较("abd", "abc")      // >0（大于）
```

### 10.4 取子串

```z
取文本("你好智语言!", 0, 3)     // "你"（字节位置 0~3）
取文本("你好智语言!", 6, 15)    // "智语言"（字节位置 6~15）
```

> 注意：起始和结束是字节位置，中文字符占 3 个字节。

### 10.5 查找

```z
查找("你好智语言", "智语言")    // 6（找到，返回起始字节位置）
查找("abc", "xyz")             // -1（未找到）
```

### 10.6 替换

```z
替换("你好世界", "世界", "智语言")   // "你好智语言"
替换("abcabc", "ab", "x")           // "xcxc"
替换("hello", "xyz", "abc")         // "hello"（未找到，原样返回）
```

## 十二、模块系统

### 12.1 模块规则

- 模块定义 ：
    ```z
    模块名:
        // 模块代码
    ```
- 模块内成员默认**私有**，外部不可访问
- 用 `公开` 关键字标记的成员可被外部访问

### 12.2 导入方式

```z
导入 "模块路径"                        // 导入整个模块，用 模块名.成员 访问
```

### 12.3 示例

模块文件 `a_module.z`：

```z
测试模块:
    公开 add(a, b):
        返回 a + b

    公开 multiply(a, b):
        返回 a * b

    内部辅助():       // 私有，外部不可访问
        返回 0
```

主文件：

```z
导入 "a_module"

主函数():
    结果1 = 测试模块.add(3, 5)           // 8
    结果2 = 测试模块.multiply(4, 5)      // 20
```

## 十六、完整示例

```z
主函数():
    输出("智语言示例程序\n")

    // 变量
    年龄 = 25
    姓名 = "小明"

    // 函数调用
    结果 = 加法(3, 5)
    输出("3+5 = %d\n", 结果)

    // 条件
    如果 年龄 >= 18:
        输出("%s 已成年\n", 姓名)
    否则:
        输出("%s 未成年\n", 姓名)

    // 循环
    求和 = 0
    计次循环 i = 0 到 10:
        求和 = 求和 + i
    输出("0到9求和 = %d\n", 求和)

    // 数组
    数组 = [10, 20, 30]
    追加(数组, 40)
    输出("数组长度 = %d\n", 取长度(数组))

    // 结构体
    人物 = {name: "悟空", age: 500}
    输出("姓名：%s 年龄：%d\n", 人物.name, 人物.age)

    退出(0)

加法(a, b):
    返回 a + b
```
